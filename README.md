# BMI CALCULATOR USING PYTHON :
BMI_Calculator

# DESCRIPTION :

    This is a simple BMI (Body Mass Index) Calculator built using Python.
    This program takes a user's name, height, and weight, calculates the BMI value, and displays the corresponding health category.
    The application also supports multiple users by allowing the user to enter the number of members. 
    It contains 2 files . BMINorm.py contains the code using function and for loop . bmiwofn.py (BMI without function) contains the code without function.

# FEATURES :

        * Calculates BMI  value using the standard BMI formula.
        * Accepts input for multiple members.
        * Displays BMI up to two decimal places.
        * Categorizes BMI into health groups according to the BMI value.
        * Simple and beginner-friendly command line program.

# CONCEPTS USED (in BMINorm.py):

              * Function.
              * Loops.
              * Conditional statements.

# HOW TO RUN :

        * Install Python on your system.
        * Save the file.
        * Then run the code.

# WORKING :
(In BMINorm.py)

         * def calculate ( x, y) - this line creates a function with parameters where x and y represent as height and weight.
         * if condition - check if x or y is less than or equal to 0.
         * else condition - calculate BMI value and its category as Under Weight , Normal Weight , Over Weight , Obese.
         * Now it takes the input from the user for number of members.
         * Now , using for loop we can iterate to get the details from the user and print the details .
         * calculate (b,c) -  calls the function to check and display the category.
         * The output shows each person's BMI Report in a clean manner.
